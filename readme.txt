============ Description ================
Psydro is a FREE worldwide online review community that brings businesses and their customers together. Using what your customers say can only benefit your business!
A Brighton based start-up, Psydro is a user-friendly platform that has been meticulously designed to help consumers make informed purchasing decisions.Dedicated to each and every consumer and business, our impartial service promises to deliver the same first-class service, whether you are a buyer, a sole trader or an international corporation.We also offer rewards in the form of prizes or experiences to consumers who regularly use the platform.We aim to keep improving the service we offer on all platforms whilst looking to keep reviewing informative as well as fun.We will be actively reviewing all feedback we get from businesses and consumers to see where and how we can improve.
Customer feedback is essential, as it educates consumers and gives them an insight into past experiences to build on purchasing confidence. It also allows businesses from any sector to follow the latest consumer trends, enabling them to gear their services more towards a current marketplace, no matter what they are selling.
In order to use Psydro review’s extension, you need a verified business merchant account. Signing up is easy and 100% free you can sign from the homepage by visiting Psydro. There are no hidden charges at all just sign up and get going.

 Features Description:
-Simple and easy to use. 
-Use the images slider to show review images on your store 
-Highly customizable from backend. Use the color picker to pick any color of your choice matching the color scheme of your store
-Customers can write reviews without leaving your store. If they do not have an account they can register from the extensions itself. 
-This extension will display 10 latest reviews and images. 
-User can give reviews by simply verifying their mobile number via OTP. 
- User can signup from extension. 
- Existing users can login using their verified email address and password to write a review.


== Installation ==

1. Upload `Psydro` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now you are ready to go :) .

= Automatic installation =
Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of Psydro Reviews, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.
In the search field type “Psydro Reviews” and click Search Plugins. Once you’ve found our review plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

= Manual installation =
Click on Plugins > add new > upload then choose the downloaded Psydro Zip file then click on install now after installation click on Activate.
Then it's ready to use :) 

= Minimum Requirements =

* PHP version 5.5 or greater 
* MySQL version 5.05 or greater  

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.

If on the off-chance you do encounter issues after an update you simply need to flush the permalinks by going to WordPress > Settings > Permalinks and hitting 'save'. That should return things to normal.
